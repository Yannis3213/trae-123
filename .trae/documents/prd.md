## 1. 产品概述

燃气服务公司月底集中处理安检工单系统，面向客服坐席、安检主管和运营负责人三类角色，覆盖入户安检、隐患整改、复查关闭三大业务模块。解决月底安检工单集中爆发时，坐席无法高效处理、状态追踪混乱、证据缺失和逾期管控不到位的问题。

- 核心价值：月底集中处理场景下，一条龙跑通安检工单从提交到复核到办结的全流程，到期预警不混队，批量处理逐条可溯
- 目标用户：燃气服务公司客服坐席（日常录入）、安检主管（审核推进）、运营负责人（终审确认）

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 客服坐席 | 角色切换选择 | 创建/提交安检工单，上传入户安检证据，查看列表和详情 |
| 安检主管 | 角色切换选择 | 审核工单，推进隐患整改，退回补正，批量处理 |
| 运营负责人 | 角色切换选择 | 终审复核，确认办结或退回，查看统计和到期预警 |

### 2.2 功能模块

1. **工单列表页**：按"待补正""复核中""办结"分组展示，支持角色筛选、到期预警（正常/临期/逾期三队分离），批量勾选和批量处理
2. **工单详情页**：入户安检、隐患整改、复查关闭三个业务Tab，每个Tab内可办理对应操作、上传证据、填写异常原因
3. **批量处理**：批量提交/审核/推进，逐条返回成功/失败原因
4. **到期预警**：正常（绿）、临期（黄）、逾期（红）三队不混，节点超时算到责任人
5. **角色切换**：页面顶部角色选择器，切换后列表和操作权限联动变化

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工单列表页 | 分组标签栏 | 待补正/复核中/办结三个Tab，显示各分组数量 |
| 工单列表页 | 到期预警区 | 正常/临期/逾期三队，颜色区分，不混合展示 |
| 工单列表页 | 工单卡片列表 | 每条显示工单号、地址、状态、到期日、当前处理人，支持勾选 |
| 工单列表页 | 批量操作栏 | 勾选后出现，支持批量提交/审核/推进，执行后逐条显示结果 |
| 工单列表页 | 角色切换器 | 顶部下拉，切换客服坐席/安检主管/运营负责人 |
| 工单列表页 | 统计面板 | 各状态数量统计，到期预警统计 |
| 工单详情页 | 入户安检Tab | 安检信息表单，上传安检照片/证据，提交/退回 |
| 工单详情页 | 隐患整改Tab | 隐患记录，整改措施，整改证据，推进/退回 |
| 工单详情页 | 复查关闭Tab | 复查结果，关闭证据，确认办结/退回补正 |
| 工单详情页 | 操作记录 | 时间线展示所有处理记录，含异常原因和补正动作 |
| 工单详情页 | 状态流转条 | 顶部显示当前工单状态和流转节点 |

## 3. 核心流程

客服坐席创建/提交入户安检工单 → 安检主管审核推进隐患整改 → 运营负责人终审确认办结。任一环节可退回补正，补正后重新进入当前环节。

```mermaid
    "安检主管审核" -->|"退回补正"| "客服坐席补正"
    "客服坐席补正" --> "安检主管重新审核"

    "运营负责人复核" -->|"退回补正"| "隐患整改"
    "复查关闭" --> "办结"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝灰（#1E293B）配暖橙（#F97316）强调色，体现燃气行业专业与安全
- 辅助色：待补正-琥珀色、复核中-蓝色、办结-绿色；逾期-红色、临期-黄色、正常-绿色
- 按钮：圆角6px，主操作用实心暖橙，次操作用描边
- 字体：系统字体栈，标题加粗，正文常规
- 布局：左侧导航+右侧内容区，卡片式列表，桌面优先
- 图标：使用 lucide-svelte 图标库

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工单列表页 | 顶部导航 | 深蓝背景，系统名称，角色切换下拉，用户头像 |
| 工单列表页 | 分组标签 | 水平Tab，带数量Badge，选中态下划线 |
| 工单列表页 | 到期预警 | 三列色块，绿色/黄色/红色，显示数量和简要信息 |
| 工单列表页 | 工单卡片 | 白色卡片，左侧状态色条，右侧操作按钮 |
| 工单列表页 | 批量操作 | 底部浮动栏，显示已选数量，批量操作按钮 |
| 工单详情页 | 状态流转条 | 水平步骤条，当前节点高亮，已完成节点打勾 |
| 工单详情页 | 业务Tab | 水平Tab切换三个模块，当前Tab内容区 |
| 工单详情页 | 操作记录 | 垂直时间线，左侧时间点，右侧操作内容 |

### 4.3 响应式

桌面优先，最小宽度1024px，侧边栏可折叠，内容区自适应宽度
