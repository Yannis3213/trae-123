## 1. 产品概述

城市公交公司月底集中处理发车计划系统，面向公交公司多岗位协作场景，实现发车计划从登记、审核到复核归档的全流程闭环管理。系统严格按角色分工推进，异常状态只允许补正或退回，杜绝越权与静默推进，确保每一笔发车计划可追溯、可审计。

- 核心价值：将月底发车计划集中处理流程数字化，消除纸质流转中的越权、遗漏和状态混乱
- 目标用户：城市公交公司发车登记员、发车审核主管、复核负责人

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登记方式 | 核心权限 |
|------|----------|----------|
| 发车登记员 | 演示账号登录 | 新建发车计划、补正退回件、查看本人发起队列 |
| 发车审核主管 | 演示账号登录 | 办理待审核计划、退回补正、查看审核队列 |
| 城市公交公司复核负责人 | 演示账号登录 | 复核归档、查看复核队列、查阅全量审计记录 |

### 2.2 功能模块

1. **发车计划队列首页**：按角色展示待办队列，旁证摘要（车辆排班、司机签到、发车确认），到期预警分组（正常/临期/逾期）
2. **发车计划详情页**：全量信息、附件、处理记录、审计备注，按当前角色展示操作按钮
3. **批量处理**：支持多选批量推进，逐条返回成功/失败原因
4. **角色切换**：顶部切换当前角色身份，列表和操作实时更新

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 队列首页 | 发车计划队列 | 按角色显示待办列表，支持筛选（状态/到期/异常），临期/逾期高亮 |
| 队列首页 | 旁证摘要面板 | 右侧展示车辆排班、司机签到、发车确认摘要卡片，点击进入详情 |
| 队列首页 | 到期预警分组 | 正常（绿）、临期（黄）、逾期（红）三组，逾期节点落到责任人 |

| 详情页 | 计划信息 | 展示发车计划完整信息、附件列表、必填证据检查 |
| 详情页 | 处理操作 | 按角色显示：提交/补正/办理/退回/复核归档，操作前校验 |

| 批量处理 | 批量操作栏 | 多选后批量推进，逐条返回成功/失败及原因 |

## 3. 核心流程

发车登记员新建发车计划 → 提交审核 → 发车审核主管办理 → 提交复核 → 复核负责人复核归档。任何节点异常只允许补正或退回，不能静默推进。

```mermaid
flowchart TD
    "发车登记员新建计划" --> "提交审核"
    "提交审核" --> "待审核"
    "待审核" --> "发车审核主管办理"
    "发车审核主管办理" --> "审核通过-提交复核"
    "发车审核主管办理" --> "退回补正"
    "退回补正" --> "发车登记员补正"
    "发车登记员补正" --> "重新提交审核"
    "重新提交审核" --> "待审核"
    "审核通过-提交复核" --> "待复核"
    "待复核" --> "复核负责人复核归档"
    "复核负责人复核归档" --> "已归档"
    "复核负责人复核归档" --> "退回补正"
```

**后端校验节点**（进入下一步前必检）：
1. 资料缺失检查：必填证据未上传则阻止推进
2. 状态冲突检查：当前状态不允许该操作则拒绝
3. 越权推进检查：当前角色无权执行该操作则拒绝
4. 超期未处理检查：超时节点标记逾期，落到责任人
5. 版本冲突检查：旧版本提交被拒绝
6. 重复提交检查：同一处理人不可重复提交

**异常处理规则**：
- 异常状态只允许「补正」或「退回」，不能悄悄推进
- 逾期批量推进时逐条拦截，返回每条失败原因
- 补正后需重新进入正常流程

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝灰（#1e293b）+ 橙色强调（#f97316）
- 辅助色：翡翠绿（正常 #10b981）、琥珀黄（临期 #f59e0b）、玫瑰红（逾期 #ef4444）
- 按钮风格：圆角 8px，主按钮实色填充，次按钮描边
- 字体：思源黑体/Noto Sans SC，标题 18px/600，正文 14px/400，辅助 12px/400
- 布局：左侧导航 + 右侧内容区，卡片式布局
- 图标：Lucide Icons 线性风格

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 队列首页 | 顶部导航栏 | 角色切换下拉、系统标题、用户信息 |
| 队列首页 | 队列列表 | 表格+行内状态标签，行点击进入详情，支持排序筛选 |
| 队列首页 | 预警分组面板 | 三色分组卡片，数字徽章，点击筛选 |
| 队列首页 | 旁证摘要 | 右侧三列小卡片，图标+数字，点击展开 |
| 详情页 | 头部信息栏 | 计划编号+状态标签+到期倒计时 |
| 详情页 | 表单区 | 分组表单，必填项红星标记，附件上传区 |
| 详情页 | 时间线 | 左侧竖线+节点圆点+时间+操作人+备注 |
| 详情页 | 操作区 | 底部固定操作栏，按角色显示按钮 |

### 4.3 响应式

桌面优先设计，最小宽度 1024px，侧边栏可折叠适配中等屏幕。
